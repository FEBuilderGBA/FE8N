##攻速
攻速計算を変えている人は要注意。このパッチを当てるとデフォルトの計算式に戻ります。
 * Atk_Spd.asm
 * Atk_Spd.gba

自分の攻速計算式が使いたい人は、asmを編集・gbaに変換してください。ASM知識必須。
