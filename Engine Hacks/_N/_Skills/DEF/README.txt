◆守備系スキル
祈り・大盾・聖盾・神盾・遠距離防御・遠距離無効・連撃防御

特記が無ければ、以下のスキルは必的か見切りで無効化出来ます

・祈り
致死ダメージを幸運x3%で耐えます
config.txtを編集することで、効果を変えられます
幸運以下のダメージを確定耐えする効果です(=ダメージと幸運が同値なら耐える)

・大盾
物理攻撃を技%でダメージ半減します

・聖盾
魔法攻撃を技%で無効化します

・神盾
全ての攻撃を半減します
相手が見切りを持っている場合でも無効化されませんが、受けるダメージは2/3です

・遠距離防御
近接攻撃以外を半減します

・遠距離無効
近接攻撃以外を無効化します
見切りや必的でも無効化できません

・連撃防御
勇者武器の効果や追撃による、2回目以降のダメージを常に半減します
発動条件が、HPが戦闘開始時未満であることなので、FEHの仕様とはやや異なります
先に攻撃をされていても、回避等によりダメージがない場合・リザイア等により戦闘開始前以上に回復した場合は発動しません
逆に先に攻撃されていなくても、デビルアクスによる自傷ダメージで発動します


処理の流れ上、拙作の連続パッチも当てないと必的が守備系スキルを貫通しなくなります。


・アイテム「おまもり」　※盾パッチも同時に当てないと正しく機能しません
アイテムの特性4の上から3番目にチェック（0x4）が入っていたら「おまもり」の効果がつきます。
効果は確実に発動する祈りです。発動したらアイテムは回数に関係無く消失します。
必的が発動していた場合は、おまもりを貫通します。
