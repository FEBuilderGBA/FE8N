奥義スキル2

武器レベルS未満では発動しない

太陽：
陽光・天空と重複設定不可
HP吸収武器では発動しない
技%の発動率
与ダメージ半分回復

流星：
武器は剣のみ
直接攻撃のみ
必殺率が0だと発動しない
与ダメージが0だと発動しない
Lv%の発動率
0.5倍～7.5倍のダメージ(命中100必殺100以上なら7.5倍で確定)
発動時に必殺モーションを取る

天空：
陽光・太陽と重複設定不可
必殺が発動している場合、発動しない
直接攻撃のみ
HP吸収武器では発動しない
Lv%の発動率
相手の守備を80%減少して攻撃、与ダメージ半分回復
発動時に必殺モーションを取る

陽光：
天空・太陽と重複設定不可
技%の発動率
相手の魔防を50%減少して攻撃、与ダメージ半分回復

撃破：
与ダメージが0だと発動しない
技%の発動率
与ダメージ2倍。必殺と重複した場合、与ダメージ4倍
